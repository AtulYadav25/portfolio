# Giftchain

A Web3 gifting platform enabling cross-chain crypto gift transfers with beautiful reveal animations.

## Overview

Giftchain is a revolutionary platform that brings the joy of gift-giving to the blockchain. Users can wrap cryptocurrency in beautiful digital packages, add personalized messages, and send them to friends and family across different blockchain networks.

## Features

- **Cross-Chain Support**: Send gifts between Solana and Sui networks
- **Beautiful Reveal Animations**: Each gift unwrap includes stunning animations
- **Personalized Messages**: Add custom messages and wrapper designs
- **NFT Wrappers**: Unique wrapper designs that can be collected
- **Social Integration**: Share gift reveals on social media

## Technical Details

The platform is built using:

- React with TypeScript for the frontend
- Solana and Sui blockchain integration
- MongoDB for user data and gift tracking
- Framer Motion for smooth animations

## Challenges

One of the main challenges was implementing cross-chain transfers while maintaining a seamless user experience. We solved this by using atomic swap protocols and providing clear status updates throughout the transfer process.
